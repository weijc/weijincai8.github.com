Android.mk总结

1.Android.mk是什么

是Android编译的时候需要用到的编译规则文件。相当于linux里的makefile。

2.Android.mk有什么用

Android.mk告诉编译系统如何编译代码。当我们使用mm进行编译时，需要Android.mk告诉编译器我的源代码是哪些，如何编译我的代码，编译完成后输出到哪一个目录下等信息。

3.Android.mk的语法简介

用一个例子来说明Android.mk的语法，这个东西不用死记硬背，看多了自然就知道了。

路径：hardware/libhardware/modules/audio
LOCAL_PATH := $(call my-dir)
include $(CLEAR_VARS)
LOCAL_MODULE := audio.primary.default
LOCAL_MODULE_PATH := $(TARGET_OUT_SHARED_LIBRARIES)/hw
LOCAL_SRC_FILES := audio_hw.c
LOCAL_SHARED_LIBRARIES := liblog libcutils
LOCAL_MODULE_TAGS := optional
include $(BUILD_SHARED_LIBRARY)
include $(CLEAR_VARS)
LOCAL_MODULE := audio_policy.stub
LOCAL_MODULE_PATH := $(TARGET_OUT_SHARED_LIBRARIES)/hw
LOCAL_SRC_FILES := audio_policy.c
LOCAL_SHARED_LIBRARIES := liblog libcutils
LOCAL_MODULE_TAGS := optional

include $(BUILD_SHARED_LIBRARY)
